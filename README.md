# it-salary-analysis-sql
SQL analysis of global IT salaries (2020-2023) to identify key market trends, regional differences, and salary benchmarks.


# Аналіз зарплат у сфері Data Science (SQL Проєкт)

## Огляд проєкту
Цей проєкт присвячений дослідженню глобальних трендів заробітних плат у сфері IT за період 2020-2023 років. Використовуючи PostgreSQL, я проаналізував понад 3700 записів, щоб виявити, як досвід, локація та масштаб компанії впливають на рівень компенсації.

## Ключові висновки (Insights)
* **Перевага середнього бізнесу:** Компанії середнього розміру (M) платять у середньому на 20% більше ($145 тис.), ніж великі корпорації ($119 тис.).
* **Гнучкість формату:** Повністю віддалені ролі залишаються надзвичайно конкурентними з середнім рівнем доходу у $136 тис.
* **Кар'єрний ріст:** Перехід з рівня Junior (EN) до Senior (SE) супроводжується зростанням зарплати майже вдвічі.
* **Топ-професії:** Менеджери з Data Science та архітектори даних є найбільш високооплачуваними ролями у даному наборі даних.

## Використані інструменти SQL
* **Агрегатні функції:** `AVG`, `MIN`, `MAX`, `COUNT`.
* **Фільтрація та групування:** `WHERE`, `GROUP BY`, `HAVING`.
* **Очищення даних:** Робота з некоректними назвами стовпців та обробка статистичних аномалій.



# IT Salary Analysis Project (SQL)

## Project Overview
This project explores global IT salary trends (2020-2023) using a dataset of 3,700+ records. I used PostgreSQL to analyze how experience, location, and company size affect compensation.

## Key Insights
* **The "Mid-Size" Premium:** Medium-sized companies pay ~20% more on average ($145k) than large corporations ($119k).
* **Remote Value:** Full-remote roles remain highly competitive, with average salaries reaching $136k.
* **Top Careers:** Data Science Managers and Data Architects are the highest-paid roles in this dataset.

## SQL Tools Used
* Aggregate functions (AVG, MIN, MAX, COUNT)
* Data filtering (WHERE, HAVING)
* Grouping & Sorting (GROUP BY, ORDER BY)
